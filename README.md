

## 🚀 Features

- **Dynamic Dashboards:** Real-time progress tracking, upcoming tasks, and AI-powered suggestions.
- **Specialized Planners:** Dedicated modules for School, UG, Daily Routines, Office work, and Exams/Study.
- **Smart Tracking:** Heatmap-style Habit Tracker and a visual "Life Path" Timeline.
- **AI Integration:** Goal-to-milestone decomposition and a personal AI planning assistant.
- **Unified Calendar:** Global view of all your events and tasks.

## 🛠️ Tech Stack

- **Frontend:** React + Vite + Vanilla CSS (Glassmorphism design)
- **Backend:** Node.js + Express (API layer)
- **Database / Auth:** Supabase (PostgreSQL)
- **Icons:** Lucide-react

## 📦 Setup & Installation

1. **Clone the repo:**
   ```bash
   git clone <repo-url>
   cd planner-app
   ```

2. **Frontend Setup:**
   ```bash
   cd frontend
   npm install
   # Create a .env file with VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY
   npm run dev
   ```

3. **Backend Setup:**
   ```bash
   cd backend
   npm install
   npm start
   ```
